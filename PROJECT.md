# AGROTRAZA NEW 1.0

## Objetivo

Desarrollar un sistema web de trazabilidad documental y operativa para empresas agroexportadoras, permitiendo controlar cada lote desde la recepción de la materia prima hasta el despacho de exportación.

---

## Estado del proyecto

**Fase actual:** Preparación de la arquitectura.

---

## Tecnologías

* React + Vite
* TypeScript
* Tailwind CSS v4
* Firebase
* Firestore
* Firebase Storage
* Firebase Authentication
* OpenAI API (futuro)
* Tesseract OCR

---

## Módulos

* Dashboard
* Gestión de Lotes
* Recepción
* Producción
* Envasado
* Producto Terminado
* Inspección
* Despacho
* Gestión Documental
* Usuarios
* Reportes
* Configuración
